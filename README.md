## ocaml

With the following code one can use ocaml withing a nix expression in a bash script. This is totally nice. ;) Simply install nix and run the shell!
```bash
#!/usr/bin/env nix-shell
(*
#!nix-shell --pure -i ocaml -p ocaml
*)
... the ocaml code
```
